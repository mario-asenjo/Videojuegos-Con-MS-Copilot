# Mi idea de videojuego

Quiero un juego sencillo de una nave espacial.

La nave está abajo y se mueve a izquierda y derecha.

Desde arriba caen meteoritos.

El jugador puede disparar para destruir meteoritos.

Cada meteorito destruido da puntos.

Si un meteorito toca la nave, se pierde una vida.

El jugador tiene 3 vidas.

Cuando se acaban las vidas, aparece una pantalla de Game Over.

Quiero que el juego funcione en navegador, en un único archivo index.html, sin instalar nada.

Los controles son:
- Izquierda y derecha para moverse.
- Espacio para disparar.
- Enter para empezar o reiniciar.

El estilo debe ser arcade, con fondo espacial oscuro.
