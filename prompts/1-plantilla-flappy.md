# Documento de diseño del videojuego

## Nombre

Flappy Copilot

## Idea general

El jugador controla un pequeño pájaro o robot volador que debe atravesar huecos entre tuberías sin chocarse.

## Tipo de juego

Arcade 2D de reflejos.

## Plataforma

Navegador web en un único archivo index.html.

## Controles

- Barra espaciadora: saltar o impulsarse hacia arriba.
- Enter: iniciar o reiniciar.

## Reglas

- El personaje cae por gravedad.
- Al pulsar espacio, sube.
- Las tuberías se mueven de derecha a izquierda.
- Si el personaje toca una tubería o el suelo, la partida termina.
- Cada tubería superada suma 1 punto.
- La dificultad aumenta poco a poco.

## Estilo visual

- Fondo azul claro.
- Suelo verde.
- Tuberías verdes.
- Personaje simple y simpático.
- Texto grande y claro.

## Requisitos

- Un único archivo index.html.
- HTML, CSS y JavaScript puro.
- Sin librerías externas.
- Sin imágenes externas.
- Debe incluir pantalla de inicio, juego y pantalla de fin.
