# Documento de diseño del videojuego

## Nombre

Quiz Express

## Idea general

Juego de preguntas rápidas donde el jugador debe responder correctamente antes de que acabe el tiempo.

## Tipo de juego

Quiz interactivo.

## Plataforma

Navegador web en un único archivo index.html.

## Reglas

- El juego muestra una pregunta y cuatro respuestas.
- Solo una respuesta es correcta.
- Cada acierto suma 10 puntos.
- Cada fallo resta 1 vida.
- El jugador tiene 3 vidas.
- Hay un temporizador de 10 segundos por pregunta.
- Cuando se acaban las preguntas o las vidas, aparece la pantalla final.

## Controles

- Clic con ratón en la respuesta.
- También permitir teclas 1, 2, 3 y 4.

## Estilo visual

- Diseño moderno tipo tarjeta.
- Colores claros.
- Botones grandes.
- Feedback visual al acertar o fallar.

## Requisitos

- Un único archivo index.html.
- HTML, CSS y JavaScript puro.
- Sin librerías externas.
- Preguntas incluidas dentro del propio código.
