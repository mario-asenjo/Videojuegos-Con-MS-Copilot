# Videojuegos con Microsoft Copilot

Repositorio de apoyo para una charla divulgativa sobre Ingeniería Agéntica e Inteligencia Artificial usando Microsoft Copilot.

El objetivo es demostrar cómo una idea sencilla de videojuego puede transformarse en:
1. Un documento de diseño en Markdown.
2. Un prompt estructurado.
3. Un videojuego web jugable en un único archivo `index.html`.

## Estructura

- `prompts/`: plantillas y prompts para usar con Microsoft Copilot.
- `demo-directo/`: espacio para guardar el juego generado durante la demo.
- `backup-meteor-defender/`: versión preparada del juego principal.
- `backup-flappy/`: backup alternativo.
